位运算

```c
& // 按位与
| // 按位或
^ // 按位异或
<< // 左移
>> // 右移
~ // 按位异或 相同返回0 不同返回1（跟0做异或保留原值，跟1做异或反转原值）
```

1. 除 ~ 以外，其他的运算符都是二目运算符
2. 参与位运算的常量或变量都是整型或者字符型

#### 常量和变量的区别

1. 常量内存空间里的值是不能改变的
2. 变量内存空间里的值是可以改变的

#### 存储单元

1. 存储单元的最小单位是字节。

32bit 机(32 位机) -> 32 根地址线 -> 2 的 32 次方(4G)

地址范围： 0 ~ 4g-1

2. 变量指向一个内存地址，

3. 类型决定这个变量占用几个内存地址

4. 大端：高位字节低地址 小端：地位字节高地址
